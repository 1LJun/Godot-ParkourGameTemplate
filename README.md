# Godot跑酷游戏模板
亲测Godot4.4.1可使用
##  项目操作说明
### 1.使用TileMapLayer搭建地图
### 2.内置两种类型的图集
#### 类型1：普通地图砖：仅起碰撞和装饰效果（竖向水流有动画）
#### 类型2：特殊地形砖：
##### 梯子1/梯子2：可向上攀爬
##### 金币：拾取加分
##### 钻石：加大量分数
##### 水砖块：带动画效果的水-装饰作用
##### 幸运方块：默认顶出钻石（可通过实例化子场景修改顶出物品：如玩家）
##### 告示牌：默认显示示例文本，按Z互动（可通过实例化子场景修改文本）
#### 3.使用实例化子场景引用更多物品（如门：用于传送至其他场景）
## 使用的插件
### Dialogue Manager（版本3.5.0）
## 游戏操作方式
### ←→进行左右移动，[SPACE]（即空格）进行跳跃和攀爬（可二段跳）
